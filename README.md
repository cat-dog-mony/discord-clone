# discord-clone

Clone discord with MERN stack.

## 백엔드

- npm
- 백엔드
- node.js
- express.js
- socket.io
- mongodb
- mongoose
- javascript (es6)

## MONGO DB 연결방법

.env에 MONGO_ID와 MONGO_PASSWORD에 몽고디비ID랑 비밀번호를 넣는다.
필요한 경우 몽고디비 url을 보고 자신의 로컬 주소에 맞게 수정한다.
